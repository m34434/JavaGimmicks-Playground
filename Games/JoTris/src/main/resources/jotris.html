JoTris - Java Open Tetris

The source code is Open Source. If you like, you can browse it here.

  • Click Start to run a new game
  • Click into field to enable keyboard control
  • Key assignments:
    • LEFT - Move to left
    • RIGHT - Move to right
    • SPACE - Turn clockwise
    • CTRL + LEFT - Turn anti-clockwise
    • CTRL + RIGHT - Turn clockwise
